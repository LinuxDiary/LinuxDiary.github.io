# rpm命令

# 安装

命令 说明
rpm -ivh PACKAGE_NAME.rpm 其中i表示安装，v表示显示安装过程，h表示显示进度
rpm -Uvh PACKAGE_NAME.rpm U表示升级安装

# 删除

命令 说明
rpm -e PACKAGE_NAME 相关依赖会一起被卸载
rpm -e –nodeps PACKAGE_NAME 不会卸载相关依赖
rpm -e –allmatches PACKAGE_NAME 删除所有跟PACKAGE_NAME匹配的所有版本的包

# 查询

命令 说明
rpm -q PACKAGE_NAME 查看包是否已安装
rpm -qp PACKAGE_NAME.rpm 获取当前目录下的rpm包相关信息
rpm -qa 列出所有已安装的软件包，可使用grep过滤结果
rpm -qf /usr/sbin/httpd 查看某个文件属于哪个软件包，可以是普通文件或可执行文件，跟文件的绝对路径
rpm -qi PACKAGE_NAME 列出已安装的这个包的标准详细信息
rpm -ql PACKAGE_NAME 列出rpm包安装后包含的所有文件
rpm -qc PACKAGE_NAME 列出rpm包安装后包含的配置文件
rpm -q –scripts kernel 列出已安装rpm包自带的安装前和安装后脚本
