# 修改密码

旧密码为空时

mysqladmin -uroot password 'NEWPASS'

旧密码不为空时

mysqladmin -uroot -pOLDPASS password 'NEWPASS'

登录修改

MySQL 5.5 & 5.6

UPDATE 命令

UPDATE mysql.user SET password=password('NEWPASS') WHERE user='root' and host='localhost';
FLUSH PRIVILEGES;

SET 命令

SET PASSWORD FOR 'root'@'localhost' = 'MyNewPass';
FLUSH PRIVILEGES;

MySQL 5.7

SET PASSWORD FOR 'root'@'localhost' = 'MyNewPass';
FLUSH PRIVILEGES;

# 重置密码

停止 MySQL 服务后，在启动时加上 --skip-grant-tables 选项，或者在 my.cnf 文件中加上 skip-grant-tables ，然后启动 MySQL ，就可以跳过授权步骤，直接登录 MySQL ，然后按照上面修改密码的方法进行密码重置。
