# Aria2 离线下载

# 安装

通过 epel 源直接安装

yum -y install aria2

安装成功后启动服务, aria2 默认监听在 6800 端口, --rpc-secret 是 RPC 秘钥, 需要自行指定, 在 AriaNg 中需要使用它, --dir 是文件下载路径

aria2c --enable-rpc --rpc-listen-all --rpc-allow-origin-all --rpc-secret=123 -c --dir /root/downloads -D

# 安装 AriaNg

AriaNg (opens new window) 是纯 HTML 和 JavaScript 的程序, 只需要一个 web 服务器即可, 这里用 Nginx , 以下是 Nginx 配置, 将 AriaNg-1.0.0.zip 下载解压后, 放置到网站根目录 html/ariang 下即可

server {
    listen 80;
    server_name example.com;
    access_log logs/ariang_access.log main;
    location / {
        root html/ariang;
        index index.html;
        auth_basic "Forbbiden";
        auth_basic_user_file /home/ec2-user/u01/nginx/conf/htpasswd;
        autoindex on;
    }
}

由于 AriaNg 没有授权认证机制, 这里简单使用 Nginx 的访问认证模块, 通过以下命令创建授权文件 /home/ec2-user/u01/nginx/conf/htpasswd , 用户名是 admin , 密码根据命令提示输入

htpasswd -c -d /home/ec2-user/u01/nginx/conf/htpasswd admin

配置完成后, 通过 sudo nginx -s reload 重载 Nginx , 然后直接通过 Nginx 中配置的域名或 IP 直接访问, 如 http://example.com

访问成功后, 在 AriaNg 设置 -> RPC(example.com...) -> Aria2 RPC 秘钥 处填入前面设置的 --rpc-secret , 然后在 Aria2 状态 处显示已连接 即可

# 使用 File Manager

File Manager (opens new window) 是一个在线文件管理系统, 支持上传下载, 文件分享, 授权访问, 在线预览等功能, 将前面 Aria2 的下载路径设置到 File Manager 的根目录下, 实现在线管理离线下载的文件

# 安装

curl -fsSL https://filebrowser.xyz/get.sh | bash

创建 File Manager 的管理目录 /root/u01/filemanager

mkdir -p /root/u01/filemanager

创建配置文件 /root/u01/filemanager/config.json

{
    "address": "0.0.0.0",
    "port": 8888,
    "database": "/root/u01/filemanager/fm.db",
    "root": "/root/u01/filemanager/rootdir",
    "log": "/root/u01/filemanager/fs.log",
    "allowEdit": true
}

通过配置文件启动

filebrowser -c /root/u01/filemanager/config.json &

然后只需要在 rootdir 下创建目录和文件即可, 通过 http://IP:8888 访问

# 防火墙

所有服务配置好以后, 在防火墙上开放 6800 80 8888 端口

iptables -I INPUT 8 -p tcp -m multiport --dport 6800,80,8888 -j ACCEPT
