# Word页码编排

# 运行环境

  • 系统环境：Windows 7
  • 软件版本：Word 2016/2013

# 分节符对页码的影响

分节符可以把一个Word文档分成多个小节，相当于分成了多个片段，可以对每个小节的页眉和页脚分别编辑不同的内容，页码就是页眉或页脚内容的一种。也就是说可以为每一小节的页眉编辑不同的logo、文字等内容，页脚也同理。

要实现这个效果的关键在于是否开启“链接到前一条页眉”选项，双击页眉或页脚时出现这个选项。

在当前小节的任一页面双击页眉或页脚时，如果开启“链接到前一条页眉”，则表示与上一小节使用相同的页眉或页脚。对于页码而言就是使用同一个页码序列。如果直接编辑页脚里的页码编号就变成了修改页脚内容（等同于插入一段相同的文字），只能删除页码，重新插入页码，不能手动编辑。

相反，如果在当前小节关闭“链接到前一条页眉”，就代表把当前小节独立出来，这样就能够在这一小节使用新的页眉和页脚。一般需要对一个Word文档中多个部分使用独立页码序列时，关闭“链接到前一条页眉”即可。

另外，Word在目录引用时就与“链接到前一条页眉”选项紧密关联，目录的页码编排与页脚的页码编排一致，也就是说只要页脚中的页码编排得当，目录就不会有问题。生成目录的时候，实际上也是一个生成页码的过程。

# 实用技巧

从文档的封面或目录之后的正文开始编排页码 然后定位到正文页面，关闭“链接到前一条页眉”，再在当前小节插入页码即可。

开启隐藏标记开关

定位到正文的上一个页面末尾，插入分节符

这个时候，当前页和下一页就已经分成2个小节了，双击页眉即可体现

在第二小节的页面双击页脚，关闭“链接到前一条页眉”，即可实现目的

关闭“链接到前一条页眉”之后，就没有“与上一节相同”字样了，代表已经与上一节的页脚独立开来，可以编辑新的内容。

注意：页眉和页脚是要分开的，当前小节的页眉并没有关闭“链接到前一条页眉”，所以此时页眉还是与上一节相同。

可以直接在第二节（正文部分）插入页码即可从1开始编码。如果在插入页码之前没有关闭“链接到前一条页眉”，那么插入的页码仍然会跨小节顺序编排，此时只需要在分节的地方关闭“链接到前一条页眉”，然后在当前小节设置页码格式，设定起始页码即可。

# 分节符的另一个作用

分节符还有一个实用的作用，就是使两个小节分别使用不同的纸张方向、纸张大小、页边距等属性，因为两个小节是独立的片段。在一个Word文档中既有普通正文又有横向表格时很实用。
